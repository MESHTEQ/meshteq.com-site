Meshteq Trust Center

Security, privacy, and operational integrity are fundamental to Meshteq’s IoT and SaaS platforms. Our systems are designed with security-by-design principles to protect industrial infrastructure, operational data, and customer systems.

Platform Security

Role-based Access Control

Meshteq platforms implement role-based access control to manage permissions across tenants and users.

Tenant Data Isolation

Meshteq SaaS applications enforce strict tenant separation using database Row Level Security (RLS), ensuring data is accessible only by authorized users.

Automated Backups

Operational data stored in Supabase is protected through automated daily backups with point-in-time recovery capabilities.

Secure Cloud Infrastructure

Meshteq applications are deployed on modern cloud infrastructure providers with built-in monitoring, network security, and encrypted communications.

Secure Payment Processing

Meshteq uses Stripe for secure billing. Payment information is never stored directly within Meshteq systems.

Vulnerability Management

Meshteq performs internal code review and dependency scanning through GitHub security tools and third-party security monitoring.

DDoS Protection

Meshteq infrastructure benefits from network-level protection through Cloudflare network protection.

IoT Network Security

IoT communication uses LoRaWAN security mechanisms including device authentication (OTAA) and encrypted network communication.

Infrastructure Providers

  • Supabase — database infrastructure and storage
  • Vercel — application hosting
  • Cloudflare — DNS and network security
  • Netmore — LoRaWAN network services
  • GitHub — source code management
  • Mailgun — transactional email services
